# Ruremai

Open ruby reference manual by browser as following.

```ruby
  Object.method(:name).rurema!
```

## Installation

Add this line to your application's Gemfile:

    gem 'ruremai'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruremai

## Usage

Method#rurema!:

```ruby
  Object.method(:name).rurema!
```

Object#mean? (shortcut):

```ruby
  Object.mean?.name
```
